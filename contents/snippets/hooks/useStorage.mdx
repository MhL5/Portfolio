They work like React state hooks, but automatically keep the stored value in sync with the corresponding storage, making it easy to save and retrieve data across page reloads or sessions.

To prevent hydration errors, you can wrap your component in a ClientOnly component or use [useIsMounted](/snippets/hooks/useIsMounted).

## Usage

```tsx
import { useLocalStorage, useSessionStorage } from "@/hooks/useStorage";
```

<br />

```tsx
// Local storage
const [name, setName] = useLocalStorage("user-name", "");

// Session storage
const [preferences, setPreferences] = useSessionStorage("user-prefs", {
  theme: "light",
  language: "en",
});

// With function as default value
const [data, setData] = useLocalStorage("cache", () => ({
  timestamp: Date.now(),
  items: [],
}));
```
